# Project-Proposal-ER-Diagram

-Deskripsi proyek
Library Borrowing Tracker adalah sistem basis data yang dirancang untuk membantu pengelolaan peminjaman dan pengembalian buku di perpustakaan. Proyek ini dikembangkan sebagai bagian dari tugas Course Project: Database Design and Implementation.

Sistem ini bertujuan untuk mempermudah pustakawan dalam mencatat transaksi peminjaman, mengelola data anggota dan buku, serta memantau status pengembalian dan denda keterlambatan.
